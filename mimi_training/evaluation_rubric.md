# 미미 검수 기준 (Evaluation Rubric) v2.0

> 미미 글쓰기 훈련의 채점 잣대를 고정한다.
> 새 세션·다른 방의 클로드도 동일 기준으로 채점하도록, 미미는 글 쓰기 전, 검수자는 채점 전 이 파일을 읽는다.
> 갱신: 2026-05-31

---

## 0. 채점 철학 (엄격 모드)

- 결함부터 본다. 칭찬은 손댈 데 없는 곳에만, 짧게.
- 지적은 단어 단위로. "약하다"가 아니라 *어느 단어가, 왜, 무엇으로 바꿀지*까지. 지적이 곧 수술 지시.
- 잘 쓴 글일수록 더 깐깐하게. "이만하면 됐지"가 천장이 된다.
- 칼같이 하되 사람은 베지 않는다. 문장을 비판하지 작가를 깎지 않는다.

---

## 1. 기본 통과 조건 (하나라도 위반 시 채점 없이 반려)

1. 첫 문장에 구체적 수치 또는 결과
2. 도구·근거 실명 (정확한 실명일 것 — 구버전·오류 금지)
3. 오늘 바로 할 수 있는 CTA 1개 (정보성 글의 경우)
4. 반복 없음 / 5. 중복 없음 / 6. 1문장 1아이디어 / 7. 능동형
8. 군더더기 부사 삭제 — 금지어: `혁명적` `압도적` `~의 시대` `매우` `정말` `아주` `확연히`. 과장·출처 없는 통계 금지.

## 2. 엔진별 점수 영역 (글 유형에 따라 100점 배분)

| 엔진 | 채점 질문 |
|------|-----------|
| 서사(감정) | "막막했다"(Tell)가 아니라 장면으로 보여주는가 |
| 지식(정보) | 사실·수치 정확한가. **틀린 사실 = 0점** |
| 설득(주장) | 명확한 주장 + 근거(Why) |
| 통합 | 두세 결이 한 줄기로 흐르는가 |
| 대화·비애 | 대사로 성격을 드러내는가. 대사 뒤 톤 해설("차갑게 말했다") 금지 |
| 서스펜스 | 단서를 한 방울씩(정보 통제) + 거리가 *좁혀지는가* |
| 1인칭 고백 | 미화·변명 0. 변명을 넣으려면 *자해*로 만들 것 |
| 순수 유머 | 코믹 시점 100% 고정 + 펀치는 하나(가장 센 걸 맨 마지막 단독) + 끝에 감동 금지 |
| 미니멀 | 감정을 감정으로 쓰지 말고 *주변·행동*으로. 마지막 한 줄이 안 쓴 99%를 떠받침 |
| 하이브리드/환희 | 솟구침은 운동으로. 정점에 정적 한 번. 연결은 두 몸의 동기화로(늦게 드러내기). 환희는 육체의 수용으로 |

---

## 3. 99점 관문 (A·B·C 전부 통과해야 99↑)

- **A. 수미상관:** 첫 문장(첫 대사)이 끝에서 같은 형태로 돌아오되 *의미가 뒤집힐 것.*
- **B. 교체 불가능성:** 임의 단어를 짚어 "바꿔도 되나?" 물을 때, *바꾸면 더 나빠지는* 단어만 남았는가.
- **C. 무의도성:** 울리려는 흔적(눈물·떨리는 목소리·"사랑한다") 0. 슬픔은 독자가 알아서 느낀다.

## 4. 합격선 진화

| 단계 | 선 | 의미 |
|------|----|------|
| 입문 | 78~85 | 기본+구조 |
| 숙련 | 92 | 기법을 의식적으로 구사 |
| 상급 | 95 | 여러 기법 통합 |
| 거장 | 98 | 기법의 *흔적*이 사라짐 |
| 완성 | 99 | 고칠 데 없음 |
| 사건 | 100 | 읽고 나서 독자가 *달라지는* 글 (채점으로 못 줌, 운명의 영역) |

## 5. 채점 출력 형식

1. 기본 조건 통과/반려 → 2. 결함부터 단어 단위 해부 → 3. 영역별 배점(+99 관문) → 4. 임의 단어 3~4개 추궁 → 5. 최종 점수 + 합격선까지 *몇 점이 왜* 모자란지

# 루나 과제 #L8 - 갭투자 금지 썸네일 (프리미엄 아트 디렉션)

## 1. 필수 레버 4가지
1. **주인공 하나, 크게:** 거대한 잠긴 자물쇠 하나가 화면을 지배. 테이프/건물 등은 뺀다.
2. **포인트 컬러 한 방:** 딥 차콜/네이비 바탕에 강렬한 빨강 하나만 포인트.
3. **깊이감 (입체감):** 스포트라이트, 그림자, 약한 비네팅 추가. (납작함 탈피)
4. **큰 타이포 자리:** 좌측 절반을 완벽히 깨끗하게 비운다.

## 2. 화풍 지정 (프롬프트 코어)
- "bold editorial poster illustration, subtle grain, soft directional shadow, limited palette (charcoal + one red + off-white)"

## 3. 하드 룰 (유지)
- 자물쇠는 반드시 잠긴 상태.
- 엔진 한계 시 16:9 (1024x576) 수동 크롭 진행.
- 이미지 내부 텍스트 렌더링 억제 (싸구려 티 방지, 후보정으로 얹기).
- 무겁고 경고성 있는 톤.

## 4. 후보정 적용
- 좌측 빈 공간에 포토샵(파이썬 PIL) 후보정을 통해 시선을 끄는 대형 텍스트 강제 오버레이.

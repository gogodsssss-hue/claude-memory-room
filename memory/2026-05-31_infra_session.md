## 2. 미미 공장 설계 (확정된 그림)
코덱스(실검·뉴스) + 주빵이(주식 팩트) → 검증된 팩트
→ 미미: 그 팩트로 글 작성 (지어내기 절대 금지)
→ 루나: 이미지
→ 워드프레스 임시저장(draft)
→ 텔레그램 알림 → 대표님 최종 검토 후 공개


## 3. 현재 상태 점검
- 재료 수집(코덱스·실검·뉴스): 작동 ✅
- 주식 팩트(주빵이): 작동(오늘 수정) ✅
- 글 작성(미미): 글솜씨 좋음, 단 사실·숫자 지어내는 약점 → SOP로 통제. **텔레그램 리스너에 미연결**(블로그/유튜브 파이프라인 소속) ⚠️
- 이미지(루나): 연결·작동 미확인 ❓
- 발행(wp_publish): 존재. **draft로 변경 필요** ⚠️
- 최종 검토: 대표님 ✅

## 4. 남은 배관 작업 (인프라 측)
1. 미미를 봇에 핸들러로 연결 (코덱스 핸들러 견본으로 복제)
2. 코덱스/주빵이 → 미미 팩트 전달 규격(인수인계 포맷) 정의
3. 루나 연결 확인
4. wp_publish 발행 상태를 publish → draft 로 변경
5. 텔레그램 승인 알림 연결

# 다중 방(뉴스·드라마·영화) 개설 런북

> 목적: 텔레그램에 주제별 방을 새로 열 때(뉴스 → 드라마 → 영화 …) 매번 처음부터 헤매지 않기 위한 재사용 플레이북.
> 전제: 봇은 **트리거 단어로 방을 구분**하고, 미미(텍스트)·루나(이미지)를 `system_prompt`로 호출하며, **명령을 보낸 방으로 자동 회신**한다.
> 비밀(토큰·키·chat_id 숫자)은 여기 적지 않는다 → 서버 `.env`에만.
> 최초 작성: 2026-06-01

## 0. 30초 요약 (TL;DR)
방 하나 = (1) 방에 **코드 봇** 넣기 + (2) **chat_id** `.env` 저장 + (3) **트리거 단어** 라우팅 + (4) **회신 대상** = 명령 온 방.
새 방 추가는 위 4개만 반복하면 된다. 봇은 항상 `.env`의 `TELEGRAM_BOT_TOKEN` 주인(`@Irsar_bot`) 하나로 모든 방을 돈다.

## 1. 핵심 원칙 (실수 방지 — telegram_bot_lessons.md 연계)
1. 방에 넣을 봇 = **코드가 쓰는 토큰의 봇** (`@Irsar_bot`). 비서 영숙(`@irir4r_secretary_bot`) 넣지 말 것. 헷갈리면 `getMe`로 토큰 주인 먼저 확인.
2. 그룹 chat_id는 마이너스(`-100…`)로 시작 → 봇을 그룹에 추가/관리자 지정한 직후 `getUpdates`로 잡는다.
3. 일반 메시지("뉴스 …" 등)를 읽으려면 **코드 봇**의 Group Privacy를 끄거나(BotFather), 코드 봇을 관리자로 지정. (영숙 거 끄지 말 것.)
4. 봇은 하나, 방은 여럿 → **트리거 단어로만 동작을 가른다.** 같은 봇이 모든 방에 있어야 한다.

## 2. 방 추가 표준 절차 (4단계)
**STEP 1 — 방에 코드 봇 초대**
  - 텔레그램에서 방 생성 → @Irsar_bot 초대 → (privacy 끄는 대신) 가능하면 관리자 지정.

**STEP 2 — chat_id 잡기**
  - TOKEN=$(grep TELEGRAM_BOT_TOKEN .env | cut -d= -f2)
  - curl -s "https://api.telegram.org/bot${TOKEN}/getUpdates" | python3 -m json.tool
  - my_chat_member 또는 message의 chat.id (-100…) 를 찾아 .env에 방별 변수로 저장.
    NEWS_ROOM_CHAT_ID=... / DRAMA_ROOM_CHAT_ID=... / MOVIE_ROOM_CHAT_ID=...
  - getUpdates에 개인 채팅만 보이면 → 토큰이 "그 방에 없는 봇" 것 (lessons 1·2번 원인).

**STEP 3 — 트리거 단어 라우팅 추가**
  - telegram_listener.py의 process_message에서 메시지 앞 단어로 분기.
  - 방별 트리거(아래 4장 표) → 해당 핸들러 → 미미/루나 system_prompt 주입.
  - 변수명 주의: 메시지 본문은 message_text (단독 message 금지 — NameError로 무응답됨, telegram-bot.md 참고).

**STEP 4 — 회신 대상 확인**
  - 출력은 **명령이 온 그 방의 chat_id**로 보낸다(고정 chat_id 하드코딩 금지). 그래야 방마다 자동으로 갈린다.

**검증**
  - pgrep -f run_bot_forever.sh | wc -l   # 1
  - pgrep -f telegram_listener.py | wc -l # 1
  - 새 방에서 트리거 보내고 → 그 방으로 회신 오는지 확인

## 3. 방별 콘텐츠 가드레일
- **공통:** 미미 페르소나 원칙(팩트 엄수, `[확인 필요]` 표시, 발행은 사람 검수 게이트). 자동 발행 금지.
- **뉴스방:** 출처/수치/날짜는 코덱스·검색·마스터가 공급. 미미는 글만. 정책·금융 주제는 면책 한 줄.
- **드라마·영화방(연예):** **공개된 사실만.**
  - 루머·미확인 열애설·사생활 추측 금지. 출연진/줄거리/공개일 등 **공식 발표·공인 매체** 기준만.
  - 인물 명예훼손·확정적 단정("~한 게 분명하다") 금지 → `[확인 필요]`로.
  - 스포일러는 명시적 경고(스포일러 주의) 뒤에만.
